## Objective
Analyze order, revenue, and user behavior data to identify growth opportunities and market performance trends.

## Dataset
This project uses a simulated food delivery dataset structured to reflect real-world Zomato order and user behavior.

## Key Analysis
- City-level and restaurant-level order and revenue performance
- New vs repeat user contribution analysis
- Order volume vs revenue comparison across markets

## KPIs Tracked
- Total Orders
- Total Revenue
- Average Order Value (AOV)
- New vs Repeat Users
- City-wise Performance

## Tools Used
- Excel (data preparation and analysis)
- Tableau (dashboard and visualization)

## Key Insights
- High-order cities do not always generate the highest revenue
- Growth is primarily driven by new user acquisition in certain markets
- Repeat users contribute a higher share of revenue despite lower volume

## Deliverables
- Interactive dashboard
- Business insights report
